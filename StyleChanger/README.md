# StyleChanger

Exercise 1: Background Color, Font Style, and Font Size Changer.

## Prerequisites

- Android Studio installed
- Android SDK and an emulator, or a physical Android device with USB debugging enabled

## Start the App

1. Open the `StyleChanger` folder in Android Studio.
2. Wait for Gradle sync to finish.
3. Click Run and choose an emulator or connected device.

## Notes

- This project is meant to be launched from Android Studio.
- The main behavior is in the activity layout and `MainActivity` Java file.
