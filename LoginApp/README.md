# LoginApp

Exercise 5: User login screen using Apache Cordova.

## Prerequisites

- Node.js installed
- Cordova CLI installed globally

## Start in the Browser

1. Open a terminal in the `LoginApp` folder.
2. Install dependencies if needed:

   ```bash
   npm install
   ```

3. Add the browser platform if it is not already present:

   ```bash
   cordova platform add browser
   ```

4. Run the app in the browser:

   ```bash
   cordova run browser
   ```

## Notes

- Default credentials are `admin` and `1234`.
- Place a PNG image at `www/img/logo.png` if you want the logo to appear.
