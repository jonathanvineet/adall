# Budget Tracker

A simple Flutter application for tracking income and expenses with category-wise breakdown.

## Features
- Add income and expense entries
- Categorize transactions
- View total income, expense, and balance
- Delete transactions
- Beautiful Material Design UI

## Tech Stack
- **Framework**: Flutter
- **Language**: Dart

## Prerequisites
- Flutter SDK installed
- Dart installed

## Setup Instructions

1. Create a new Flutter app:
```bash
flutter create app
cd app
```

2. Replace the content of `lib/main.dart` with the code from `main.dart`

3. Run the app:
```bash
flutter run -d chrome
```

## How to Use
1. Click the floating action button (+) to add an entry
2. Enter the category name and amount
3. Toggle between Income and Expense
4. Click Save to add the entry
5. View the summary at the top showing total income, expense, and balance

## Result
The application successfully tracks income and expenses with real-time balance calculation.
