# Email and Notifications

Exercise 3: Send Email and show a notification using Android Studio, JavaMail, and NotificationManager.

## Prerequisites

- Android Studio installed
- Android SDK and an emulator, or a physical Android device with USB debugging enabled
- A Gmail account with an App Password if you want to test SMTP sending

## Start the App

1. Open the `mail` folder in Android Studio.
2. Sync Gradle so the JavaMail dependencies download correctly.
3. Replace the placeholder email credentials in `MainActivity.java` with test values.
4. Click Run and install the app on an emulator or connected device.

## Notes

- Add the required internet and notification permissions in the manifest.
- Do not hardcode real credentials in production builds.
