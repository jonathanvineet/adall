# Calculator

Exercise 2: Calculator application using Android Studio and Java event listeners.

## Prerequisites

- Android Studio installed
- Android SDK and an emulator, or a physical Android device with USB debugging enabled

## Start the App

1. Open the `calculator` folder in Android Studio.
2. Wait for Gradle sync to complete.
3. Click Run and launch the app on an emulator or connected device.

## Notes

- This project is designed for Android Studio, not browser execution.
- Use the number keys and operators in the calculator screen to test the logic.
