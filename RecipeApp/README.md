# RecipeApp

Exercise 7: Recipe suggester with nutrition information using Ionic and Angular.

## Prerequisites

- Node.js installed
- Ionic CLI installed globally

## Start in the Browser

1. Open a terminal in the `RecipeApp` folder.
2. Install dependencies if needed:

   ```bash
   npm install
   ```

3. Start the app in the browser:

   ```bash
   ionic serve
   ```

## Notes

- The app is intended to run in the browser during development.
- If `ngModel` or template directives are used, make sure the required Angular imports are present in the page module.
