# Library Management

Exercise 4: Library management app using Android Studio and SQLite.

## Prerequisites

- Android Studio installed
- Android SDK and an emulator, or a physical Android device with USB debugging enabled

## Start the App

1. Open the `library` folder in Android Studio.
2. Wait for Gradle sync to finish.
3. Click Run and start the app on an emulator or connected device.

## Notes

- The database is created automatically the first time the app launches.
- Sample data is seeded on first run, so the list buttons will show results immediately.
